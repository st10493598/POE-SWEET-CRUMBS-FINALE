# Sweet Crumbs Bakery

## Description

Sweet Crumbs Bakery is a responsive website for a local bakery located in Thohoyandou, South Africa. The website showcases the bakery's delicious baked goods, provides information about the team, displays a menu of available items, allows customers to place orders, features a gallery of products, and includes contact details.

## Version History

The first version of this website was a basic HTML website. The current version is an updated one with CSS for improved styling and responsiveness.

## Features

- **Home Page**: Hero section, featured products (cupcakes, custom cakes, muffins), customer testimonials, and call-to-action for ordering.
- **About Us**: Information about the bakery and its team.
- **Menu**: Detailed list of baked goods and pricing.
- **Order**: Online ordering form for customers.
- **Gallery**: Visual showcase of bakery products and team.
- **Contact**: Contact information and form for inquiries.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Playfair Display and Lato)

## Installation

To set up and view the website locally:

1. Clone or download the project repository to your local machine.
2. Navigate to the project directory (e.g., `Sweet-Crumbs-Bakery`).
3. Open the `index.html` file in your preferred web browser (e.g., Chrome, Firefox, or Edge).
4. No additional dependencies or installations are required, as this is a static website.

## Usage

Once opened in a browser, navigate through the different pages using the menu in the header. The website is fully responsive and optimized for desktop, tablet, and mobile devices.
