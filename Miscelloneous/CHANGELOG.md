# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-01

### Before
- Initial release of Sweet Crumbs Bakery website as a basic HTML website
- Home page with hero section, featured products, and testimonials
- About, Menu, Order, Gallery, and Contact pages
- Navigation menu and footer
- Contact form and ordering functionality

## [2.0.0] - 2023-10-15

### Added
- Updated website with CSS styling for improved appearance
- Responsive design for mobile and desktop devices
- Integration of Google Fonts (Playfair Display and Lato)
- Enhanced navigation menu and footer styling
